
# HuggingFace LLM Access using LangChain

This repository contains my learning experiments while accessing open-source LLMs using HuggingFace and LangChain.

## What I Practiced
- Connecting to HuggingFace models
- Using LangChain LLM wrapper
- Testing prompt responses
- Exploring chat vs regular LLM behavior

## Tools Used
- Python
- LangChain
- HuggingFace Hub
- Google Colab
